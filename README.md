# GraphQL 

## Correr servidor localmente

```
cd graphql-server
```
```
npm install
```
```
DBUSER=xxxxxx DBPSW=xxxxxx npm start
```

## Correr cliente localmente

```
cd relay-client
```
```
bash start.sh
```
